CSS Width 和 Height 屬性教學

Width（寬度）和 Height（高度）是用來控制元素大小的基本 CSS 屬性。

1. Width（寬度）控制

Width 決定元素的寬度，可以用像素（px）或百分比（%）來設定。

寬度 100px
寬度 200px
寬度 400px
寬度 50%

2. Height（高度）控制

Height 決定元素的高度，通常用像素（px）來設定。

高度 50px
高度 100px
高度 150px

3. 同時設定 Width 和 Height

可以同時設定寬度和高度來完全控制元素的大小。

寬 200px × 高 100px
正方形 150px × 150px
