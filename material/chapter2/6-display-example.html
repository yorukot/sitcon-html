CSS Display 屬性教學

Display 是 CSS 中最重要的屬性之一，它決定了元素如何顯示在網頁上。

1. display: block (區塊元素)

區塊元素會佔滿整行的寬度，並且會換行顯示。

我是區塊元素 1
我是區塊元素 2
我是區塊元素 3

2. display: inline (行內元素)

行內元素不會換行，會並排顯示在同一行。

行內元素 1 行內元素 2 行內元素 3

3. display: inline-block (行內區塊元素)

結合了 inline 和 block 的特性，可以並排顯示，但也可以設定寬度和高度。

行內區塊 1
行內區塊 2
行內區塊 3

4. display: none (隱藏元素)

元素會完全隱藏，不佔用任何空間。

顯示的元素
隱藏的元素（你看不到我）
另一個顯示的元素

5. display: flex (彈性盒子)

讓子元素可以彈性排列，是現代網頁布局的常用方式。

彈性盒子 1
彈性盒子 2
彈性盒子 3

重點整理

  • block：佔滿整行，會換行
  • inline：並排顯示，不能設定寬高
  • inline-block：並排顯示，可以設定寬高
  • none：完全隱藏元素
  • flex：彈性布局，適合排列多個元素
